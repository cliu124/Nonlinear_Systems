oclib index

  • Tcbcf: boundary conditions, free T case
  • Tcbcfe: extended boundary conditions, free T case
  • Tcbcjac: Jac of boundary conditions, free T case
  • Tcbcjace: Jac of extended boundary conditions, free T case
  • Tfjac: Jac of f for CPs, free T case
  • Tfjace: Jac of extended f for CPs, free T case
  • Tmrhs: rhs for CP-ODE, free T case
  • bvphdw: custom Newton solver for bvps occuring for canonical paths
  • cpsolver: wrapper for solving canonical-path problems by mtom or bvphdw
  • disjcaT: discounted current value jca
  • floqpsmatadj: compute multipliers by pqzSchur; here also return unstable
  • getPsi2: get matrix Psi for projecting out E_u of target CSS
  • isc: initial state continuation to compute canonical paths to a CSS or CPS,
  • jca: integrate J_c in space and normalize by p.vol
  • jcaiT: jca along canonical path, with dependence
  • ocbra: standard output to bifurcation diagram for OC
  • ocinit: set parameters for comp.of CPs to CSS/CPS
  • ocstanopt: set standard options for OC problems
  • pqzschur_reorder_adj:
  • psol3DT: OC version of x-t plots, with T dependence
  • spcalcoc2: OC version of spcalc, calculate (gen) EVals/EVecs of Gu'.
  • tadev: return target deviation in sup and l2 norm
  • vparscal: scaling of parameters for norm, used in isc
  • wrapbc: mtom-wrapper for CP BCs, free T case
  • wrapbce: mtom-wrapper for CP extended BCs, free T case
  • wrapbcjac: mtom-wrapper for Jac of CP BCs, free T case
  • wrapbcjace: mtom-wrapper for Jac of CP extended BCs, free T case
  • wrapf: mtom-wrapper for rhs for CPs, free T case
  • wrapfjac: mtom-wrapper for Jac of rhs for CPs, free T case
  • wrapfjace: mtom-wrapper for Jac of extended rhs for CPs, free T case
