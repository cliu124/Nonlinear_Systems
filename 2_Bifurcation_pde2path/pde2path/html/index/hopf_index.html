hopf index

  • flmcoll: extract blocks from Hopf-jac to prepare multiplier-comp
  • floq: compute Floquet multipliers and instability index of u_H
  • floqap: Floquet multipliers and ind(u_H) (a posteriori)
  • floqps: Floquet multipliers by periodic Schur decomposition
  • floqpsap: Floquet multipliers by pqzSchur (a posteriori)
  • fsol_np: interface to call fsolve for Hopf in natural parametrization
  • fsolextfh: rhs for fsolve
  • fsolexth: interface to call fsolve for Hopf in arclength parametrization
  • gethoA: build extended Jac for Hopf-arclength setting
  • gethoA_np: build Jac for Hopf natural parametrization setting
  • getmuc: get critical multiplier ga, i.e. ga closest to unit circle
  • getqhder: get derivatives of Hopf constraint p.fuha.qfh
  • hoMini: initialize M for TOM (para=3); also used for para=4 with nq=0,
  • hoarcl: arclength for Hopf
  • hoasol: solve in arclength setting but with ds=0, use to decrease tol
  • hoassempbc: assemble rhs and jac for hopf with pBC, pad y accordingly
  • hobc: BC for Hopf, for tomsol via tom
  • hobcjac: BC jacobian for Hopf, for tomsol via tom
  • hobifdetec: Hopf version of bifdetec, para=4 required
  • hobra: output for bifurcation diagram, setup for Hopf, including Krylov
  • hobratw: output for bifurcation diagram, setup for TW-Hopf
  • hocont: cont for Hopf; also called by cont if p.sol.ptype=3
  • hodummybc: dummy BC for Hopf, used (and then irrelevant) in tomassempbc
  • hodummybcjac: dummy BC jac for Hopf, used (but irrelevant) in tomassempbc
  • hogetnf: get dlam and alpha for Hopf initial guess
  • hogradinf: print info on u'_infty
  • hoinistep: tomsol for first 2 points on branch; also comp.secant (para=3)
  • hojac: Jac of rhs for Hopf (para=3)
  • honloop: Newton loop for Hopf at fixed lam, for now nqh=0
  • honloop_np: Newton loop for Hopf in natural parametrization (fixed lam)
  • honloopext: Newton loop for Hopf in arclength
  • honorm: arclength norm (with weights) for Hopf
  • hopc: phase condition
  • hopftref: mesh refinement in t for hopf;
  • hoplot: basic plotting routine for Hopf orbits
  • horhs: rhs for Hopf, with aux vars for pBC in t
  • hosjac: Jac for Hopf, pad u with par, then call getGu
  • hosrhs: pad u with par, then call resi
  • hosscon: Hopf step size control
  • hostanheadfu: standard "header function" for runtime output for Hopf
  • hostanopt: standard options for Hopf (TOM)
  • hostanparam: settings of p.hopf parameters to 'standard' values
  • hostanufu: standard "user function" called after each cont.step, HOPF
  • hoswibra: branch switching from Hopf points
  • hoswipar: switch parameter for (Hopf) solution from file
  • hoswiparf: switch parameter for (Hopf) solution from file (old name)
  • hotintxs: time integration with LU-decomp. of M+dt*K,
  • huassem: assemble rhs and jac for hopf (arclength), extracted from TOM
  • huassempbc: assemble rhs and jac for hopf with pBC, pad y accordingly
  • hunorm: l2-norm, dim-independent
  • lframeplot: x-t-plot of rel. Hopf orbit with period T in lab frame
  • lssfloq: AMG interface for it. soln of LSS for Floq-comp (eperimental!)
  • pobifpred: predictor at bif FROM hopf orbit via monodromy M
  • poiniguess: do iniguess for periodic orbit (indep. of branch-switching)
  • poswibra: branch-switching from periodic orbits (pitch/trans or period-doubling),
  • sety0dot: set y0dot for the phase condiion (in t) for Hopf orbits
  • twswibra: branch switching from Hopf points to TWs in O(2) symmetry
  • uhopftref: uniform mesh refinement in t for hopf ;
