spcont index

  • bpcontexit: exit BP continuation
  • bpcontini: initialization for BP continuation
  • bpcresi: residual for extended system for BP continuation
  • bpjaccheck: compare numerical and assembled pa_u(G_u phi) (for BP continuation)
  • bploc: localize BP via extended system
  • bpreduce: reduce to original system in case of BPcontinuation
  • fpcresi: residual for extended system for FP continuation
  • fploc: localize foldpoint via extended system
  • getGuTpsiu: \pa_u(G_u'*\psi) via numjac
  • getGubpco: BP continuation jacobian
  • getGufoco: FP continuation Jacobian
  • getGuhpco: HP continuation Jacobian
  • getGuphih: \pa_u(G_u'*\psi) via numjac
  • getGuphiu: \pa_u(G_u'*\psi) via numjac
  • hpcbra: output for HPC, in particular frequency om
  • hpcontexit: exit Hopf point continuation
  • hpcontini: initialization for Hopf-point continuation
  • hpcresi: residual for extended system for HP continuation
  • hpjaccheck: compare numerical and assembled pa_u(G_u phi) (for HP cont)
  • hploc: localize Hopf point via extended system
  • hpreduce: reduce to original system in case of Hopf continuation
  • resiGuTpsi: interface for numjac for \pa_u(Gu'*psi)
  • resiGuphi: interface for numjac for \pa_u(Gu*phi)
  • resiGuphih: interface for numjac for \pa_u(Gu*phi)
  • spcontexit: exit spectral continuation; p.sw.spcont~=0 is required
  • spextend: extend system with spectral part in case of spectral continuation
  • spjaccheck: compare numerical and assembled pa_u(G_u phi) (for foldcont)
