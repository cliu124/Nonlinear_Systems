linalg index

  • bel: bordered elimination, called by lssbel
  • beliluon: convenience SCRIPT to switch on bel (and lssAMG) in problem p
  • belon: switch on bel (bordered elimination) in p
  • belpi: bordered elimination with post iterations
  • blss: BorderedLinearSystemSolver
  • blssbel: Increases p.bel.bw and calls lssbel.
  • blsslu: check for LU in p.LU, and use that if up to date, otherwise update
  • bss: bordered system solution after govaerts
  • gcafun: solver to be called in eigs; calls SM solver gclsseigs
  • gcafunilu: solver to be called in eigs; calls ilueigs
  • gcblss: customized blss for GC
  • gclss: customized LinearSystemSolver for global coupling.
  • gclssdefl: customized LinearSystemSolver for global coupling.
  • gclsseigs: customized LinearSystemSolver for GC, eigs-version
  • getker: get kernel of G_u, used in q(c)swibra, or as prep for gentau
  • getspec: get (and plot) spectrum of lin around p.u
  • ilueigs: AMGsolver for eigs-version
  • ilupparam: checks entries in p.ilup and resets amgopt in lssAMG
  • initeig: init eigref=guesses for imag parts of evals
  • initwn: set b vectors to prepare initeig (winding nr comp)
  • lss: default LinearSystemSolver, i.e. \
  • lssAMG: interface to use ilupack for solving Ax=b. (AMG or ILU version)
  • lssAMGb: Uses ilupack (www.icm.tu-bs.de/~bolle/ilupack/) for solving Ax=b.
  • lssILUb: Uses ilupack4m (https
  • lssbel: bordered elimination lss after govaerts with post iterations and
  • lssdefl: customized LinearSystemSolver for deflation
  • lsslu: check for LU in p.LU, and use that if up to date
  • lsslueigs: check for LU in p.LU, and use that if up to date
  • mbel: m-border (in p.nc.mbw) border elimination LSS
  • mbellu: m-border (in p.nc.mbw) elimination LSS, using lu-fact.
  • myeigs: interface for eigs, controlled by p.sw.eigssol
  • setbel: convenience function for switching on bel (bordered elimination)
  • setbelilup: convenience function for switching on bel with AMGsolver
  • setilup: convenience function for setting AMGsolver (ilupack) parameters
  • spcalc: compute p.nc.neig(j) EVals of Gu near p.nc.eigref(j), j=varargin
  • stanqM: standard extension (by zero borders) of M for q (constraints)
  • vspcalc: loop over shifts and call spcalc
