plot index

  • bdmov: sample SCRIPT for creating movies moving through a BD
  • breakyaxis: from WEB
  • coneplot: plot on surface of cone
  • faceplot: plot u on faces of 3D domain
  • fouplot: plot of FFT of p.u; currently in 1D and 2D; k-range as aux-arg
  • freezeColors: Lock colors of plot, enabling multiple colormaps per figure.
  • homov2d: create movie from hopf-orbit in p.hopf.y, TEMPLATE
  • homov3d: create movie from hopf-orbit in p.hopf.y, TEMPLATE
  • hoplotf: call hoplot for point read from dir/pt
  • hospatplot: plot "Hopf vector" at bif (contours for 2 compo-sys if pstyle=0)
  • hotanplot: plot Hopf tangent
  • huafixed: hack of MATLAB's annotation, use in plotbra with fancy=2
  • isoplot: plot isolevels for u (3D)
  • levplot: plot level-lines u1=cl
  • mymov2avi: HU-interface to VideoWriter (which seems strange under linux!)
  • nola: convenience (script) to switch off labels
  • nolab: convenience (script) to switch off labels
  • nolti: convenience (script) to switch off labels and ticks
  • noltix: convenience (script) to switch off labels
  • noticks: convenience (script) to switch off ticks
  • p2pc: convenience function to choose colors.
  • plotEvec: plot real part of selected eigenvector from spcalc computation.
  • plotbra: plot branch; many possible argument lists
  • plotbradat: plot branch component vs. branch component
  • plotbraf: Plot branch from p-structure file with plotbra
  • plotfloq: plot Floquet multipliers in p (p2p-struct, or directory)
  • plotsl: plot slice of u through grid.
  • plotsol: plot component of p.u in struct p; many possible argument lists
  • plotsolf: legacy interface to plotsol
  • plotsolu: plot component of u with plotsol
  • plotspec: plot spectrum of G_u
  • plottauf: plot tangent from dir/sfname.mat
  • rwplot: plot of RW computed via twswibra
  • screenlayout: open, clear and position windows for sol, branch and info
  • slplot: slice plot (3D)
  • tintplot1d: plot output from time integration
  • tintplot2d: plot output from time integration
  • tri2grid: interpolate (2D) u from triangulation to grid (obsolete ?)
  • tricontour: level-lines for Delaunay; by D.C. Hanselman, University of Maine
  • twplot: x-t-plot of TW computed via twswibra
  • unfreezeColors: restore after freezeColors, by John Iversen
  • xtplot: plot 1+1 dim soln
