pdeo index

  • diskpdeo: disk pde object, arguments R and nphi (#of points on circle)
  • diskpdeo2: alternative disk pdeo, generates radially symmetric mesh
  • ellpdeo: ellipse pdeo
  • freegeompdeo: free geometry, x,y=bd-coordinates, hmax=max mesh-width
  • grid2DHU: (classdef) put into grid 2D, i.e., here for compatibility
  • hexpdeo: domain and mesh for a unit hexagon
  • psqpdeo: domain and mesh for a perturbed square (rounded corners)
  • secpdeo: sector pde object, with radial mesh, based on freeGeoPts
  • secpdeo1: sector pdeo contructor, simple meshing
  • spherepdeo: (preimage-)domain and mesh for a sphere
  • stanpdeo1D: standard 1D OOPDE object for p2p (classdef)
  • stanpdeo1Db: 1D pdeo, x\in(l1,l2) (not x\in(-lx,lx) as in stanpdeo1D)
  • stanpdeo2D: standard 2D OOPDE object for p2p (classdef)
  • stanpdeo2Db: standard 2D OOPDE object for p2p (classdef)
  • stanpdeo3D: standard 3D OOPDE object for p2p (classdef)
  • tripdeo: domain and mesh for an equilat. triangle
  • wankelpdeo: domain and mesh for an equilat.triangle with rounded corners
